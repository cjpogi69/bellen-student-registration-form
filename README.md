# Student Registration Form
 
## Description
A web-based registration system developed using HTML, Google Apps Script, and Google Sheets.
 
## Features
- Collects student information
- Stores submissions automatically in Google Sheets
- Uses Google Apps Script for backend processing
- Hosted using GitHub Pages
 
## Technologies Used
- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets
- GitHub
